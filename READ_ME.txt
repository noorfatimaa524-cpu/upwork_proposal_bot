# 🤖 LangGraph AI Agent (Python + OpenAI)

This project is a Python-based AI Agent built using **LangChain, LangGraph, and OpenAI GPT-4o-mini**.  
It works as an interactive CLI assistant that can also use tools like saving files and formatting proposals.

---

## 🚀 Features

- AI Chat Agent using GPT-4o-mini
- LangGraph ReAct-based architecture
- Custom Tools:
  - Save AI output to file
  - Format freelancer proposals
- CLI-based chatbot interface
- Secure API key handling using `.env`
- Easy to extend for CRM or automation systems

---

## 🧠 Tech Stack

- Python 3.10+
- OpenAI API
- LangChain
- LangGraph
- python-dotenv
